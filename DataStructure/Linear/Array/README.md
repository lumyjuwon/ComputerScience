# Array

### Selection Sorting

Selection Sorting은 정렬하고자 하는 Array와 Array내에서 SubArray라는 가상의 영역을 둬 Array내에서 가장 작은 값을 SubArray에 연속으로 배치하는 형식의 Sorting이다.
